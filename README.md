# Trang tin tức đơn giản

Trang tin cho phép đăng bài, xét duyệt và phân quyền người dùng.

## Môi trường phát triển

**Yêu cầu:** Nodejs 

# Install dependences

Server & Client : npm install

### Run project

Server : node server (npm start if using nodemon)

Client : npm start
